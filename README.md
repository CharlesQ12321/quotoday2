# 每日一签 (Quotoday)

![每日一签](生成读书软件图标.png)

一款专为读书爱好者设计的Web应用，帮助用户轻松创建、管理和分享书籍中的精彩片段。通过AI文字识别技术，快速将纸质书籍内容转化为电子书签，并生成可分享的精美图片。

## 🌟 核心功能

### 📷 智能文字识别
- 通过拍照或上传图片自动识别文字内容
- 支持多种字体和排版格式
- 实时显示识别进度和状态
- 提供文字编辑功能，可手动修正识别结果
- 支持图片旋转功能，提高识别准确度

### 🎨 图片编辑
- 对上传的图片进行裁剪编辑，选择有效的文字区域
- 实时预览编辑效果
- 支持鼠标和触摸操作
- 提供直观的裁剪框和调整手柄

### 📚 书签管理
- 查看、搜索、过滤和管理已保存的书签
- 支持按内容、书名、作者和标签搜索
- 按标签、书名、作者快速筛选书签
- 查看书签详情，编辑和删除书签
- 按时间顺序显示所有书签

### 🏷️ 标签管理
- 创建、编辑和删除书签标签
- 显示每个标签的使用次数
- 通过标签快速筛选书签
- 点击标签直接跳转到首页并应用筛选

### 💾 数据导入导出
- 将书签数据导出为JSON文件
- 从JSON文件导入数据
- 自动合并现有数据
- 支持清除所有数据（谨慎操作）

### 📤 书签分享
- 将书签生成为PNG格式的图片
- 支持使用系统分享功能分享
- 支持下载图片到本地
- 包含书签的完整信息和选定的设计模板

### 🎭 UI风格切换
- 支持切换不同的UI风格：简约、文艺、暗黑
- 动态加载不同的CSS样式文件
- 保存用户的风格偏好设置
- 实时切换，无需刷新页面

## 🛠️ 技术实现

### 前端技术栈
- **HTML5**：页面结构和语义化标签
- **CSS3**：样式设计和动画效果
- **JavaScript (ES6+)**：交互逻辑和功能实现
- **Tailwind CSS**：实用优先的CSS框架，用于快速构建响应式界面
- **外部库**：
  - html2canvas.min.js：用于生成书签图片
  - jspdf.min.js：用于PDF导出功能

### AI文字识别
- **API**：使用GLM-4.6V API进行文字识别
- **实现方式**：
  1. 将图片转换为Base64格式
  2. 构建API请求并发送到GLM-4.6V
  3. 处理API响应并提取识别结果
  4. 对识别结果进行后处理，修正常见错误

### 数据存储
- **存储方式**：使用localStorage进行本地存储
- **存储结构**：
  - 书签数据：包含id、书名、作者、内容、标签等信息
  - 标签数据：包含id、名称、使用次数等信息
  - 设置数据：包含用户偏好设置，如UI风格

## 🚀 快速开始

### 安装和使用
1. **克隆或下载项目**到本地
2. **打开应用**：在浏览器中打开 `index.html` 文件
3. **创建书签**：点击底部导航栏的"创建"按钮
4. **上传图片**：拍照或选择包含文字的图片
5. **裁剪区域**：调整虚线框选择文字的有效区域，点击"确认区域"按钮
6. **编辑信息**：填写书名、作者、页码等信息
7. **添加标签**：为书签添加相关标签
8. **保存书签**：点击"保存书签"按钮完成创建

### 管理书签
1. **查看书签**：在首页浏览所有书签
2. **搜索书签**：使用顶部搜索栏搜索特定内容
3. **筛选书签**：使用标签、书名、作者下拉菜单筛选相关书签
4. **查看详情**：点击书签查看详细信息
5. **编辑书签**：在书签操作菜单中点击"编辑"按钮修改信息
6. **删除书签**：在书签操作菜单中点击"删除"按钮移除书签
7. **分享书签**：在书签操作菜单中点击"分享"按钮生成分享图片

### 管理标签
1. **进入标签页**：点击底部导航栏的"标签"按钮
2. **创建标签**：点击"+"按钮，输入标签名称，点击"确定"按钮
3. **编辑标签**：点击标签的"编辑"按钮修改信息
4. **删除标签**：点击标签的"删除"按钮移除标签
5. **使用标签**：点击标签直接跳转到首页并应用该标签筛选

### 数据备份
1. **导出数据**：进入"设置"页面，点击"导出数据"按钮
2. **导入数据**：进入"设置"页面，点击"导入数据"按钮，选择备份文件
3. **清除数据**：进入"设置"页面，点击"清除数据"按钮（谨慎操作）

### 切换UI风格
1. **进入设置页**：点击底部导航栏的"设置"按钮
2. **选择风格**：在"UI风格" section中选择喜欢的风格（简约、文艺、暗黑）
3. **应用风格**：系统自动应用所选风格并保存设置

## 🎯 应用场景

### 阅读笔记
- **场景**：阅读纸质书籍时，遇到精彩片段
- **使用方式**：拍照识别内容，添加到书签，记录感悟
- **价值**：快速保存阅读心得，避免遗漏重要内容

### 学习资料整理
- **场景**：学习过程中，需要整理重点内容
- **使用方式**：识别教材内容，添加标签分类，方便复习
- **价值**：系统化整理学习资料，提高学习效率

### 内容分享
- **场景**：发现精彩内容，想分享给朋友
- **使用方式**：创建书签，选择美观模板，生成图片分享
- **价值**：提升分享体验，增加内容传播效果

### 个人知识库
- **场景**：长期积累阅读内容，构建个人知识体系
- **使用方式**：持续添加书签，使用搜索和标签管理
- **价值**：形成个人知识储备，方便随时查阅

### 夜间阅读
- **场景**：夜间阅读时，希望减少屏幕亮度对眼睛的刺激
- **使用方式**：切换到暗黑模式，舒适阅读
- **价值**：保护眼睛，提供舒适的阅读环境
