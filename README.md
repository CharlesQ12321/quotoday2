# 每日一签 (Quotoday)

![每日一签](生成读书软件图标.png)

一款专为读书爱好者设计的Web应用，帮助用户轻松创建、管理和分享书籍中的精彩片段。通过AI文字识别技术，快速将纸质书籍内容转化为电子书签。

## 🌟 核心功能

### 📷 智能文字识别
- 支持拍照或从相册/文件上传图片
- 图片裁剪与旋转功能，提高识别准确度
- 基于GLM-4V API的智能文字识别
- 自动将识别结果填入书签内容

### 📝 书签创建与管理
- 创建书签：填写书名、作者、内容、备注和标签
- 智能输入：书名和作者支持下拉选择历史记录
- 文本处理：标点符号转换（中英文互换）、取消分段、内容查重
- 标签管理：支持多标签，可新建或选择已有标签

### 🔍 书签浏览与筛选
- 首页展示所有书签，按时间倒序排列
- 搜索功能：支持按内容、书名、作者搜索
- 多维筛选：按标签、书名、作者快速筛选
- 书签展开：点击书签展开查看完整内容和操作按钮

### 🏷️ 标签管理
- 标签列表：显示所有标签及使用次数
- 标签操作：创建、编辑、删除标签
- 快速筛选：点击标签直接跳转到首页并应用筛选

### 💾 数据导入导出
- 导出方式：导出为JSON文件或文本框复制
- 导入方式：从JSON文件导入或文本框粘贴
- 数据合并：自动合并现有数据，避免重复
- 数据清除：支持清除所有数据（谨慎操作）

### 📤 书签分享与查看
- 书签查看：全屏查看书签，支持左右滑动切换
- 图片生成：将书签生成为PNG格式图片
- 图片下载：支持下载书签图片到本地

### 🎨 多风格UI
- 三种风格：简约、鎏金、暗黑模式
- 实时切换：无需刷新页面即可切换风格
- 偏好保存：自动保存用户的风格选择

## 🛠️ 技术实现

### 前端技术栈
- **HTML5**：页面结构和语义化标签
- **CSS3**：样式设计和动画效果
- **JavaScript (ES6+)**：交互逻辑和功能实现
- **Tailwind CSS**：实用优先的CSS框架
- **Font Awesome**：图标库
- **外部库**：
  - html2canvas：用于生成书签图片
  - jspdf：用于PDF导出功能

### AI文字识别
- **API**：使用GLM-4V API进行文字识别
- **实现流程**：
  1. 用户上传图片或拍照
  2. 图片压缩和裁剪处理
  3. 转换为Base64格式
  4. 调用GLM-4V API进行识别
  5. 提取识别结果并填入表单

### 数据存储
- **存储方式**：使用localStorage进行本地存储
- **数据结构**：
  - 书签数据：id、title（书名）、author（作者）、content（内容）、note（备注）、tags（标签ID数组）、created_at（创建时间）
  - 标签数据：id、name（名称）、count（使用次数）
  - 设置数据：style（UI风格）

## 🚀 快速开始

### 安装和使用
1. **克隆或下载项目**到本地
2. **打开应用**：在浏览器中打开 `index.html` 文件
3. **创建书签**：
   - 点击底部导航栏的"创建"按钮
   - 点击上传区域，选择拍照或从相册/文件选择
   - 调整裁剪区域，点击"确认区域"进行文字识别
   - 填写书名、作者等信息
   - 添加标签（至少一个）
   - 点击"保存书签"完成创建

### 管理书签
1. **查看书签**：在首页浏览所有书签
2. **搜索书签**：在顶部搜索栏输入关键词
3. **筛选书签**：使用标签、书名、作者下拉菜单筛选
4. **展开书签**：点击书签项展开查看完整内容和操作按钮
5. **编辑书签**：点击"编辑"按钮修改信息
6. **删除书签**：点击"删除"按钮移除书签
7. **查看详情**：点击"查看"按钮全屏查看书签

### 管理标签
1. **进入标签页**：点击底部导航栏的"标签"按钮
2. **创建标签**：点击"+"按钮，输入标签名称
3. **编辑标签**：点击标签的"编辑"按钮
4. **删除标签**：点击标签的"删除"按钮
5. **使用标签**：点击标签跳转到首页并应用筛选

### 数据备份
1. **导出数据**：进入"设置"页面，点击"导出数据"，选择导出方式
2. **导入数据**：进入"设置"页面，点击"导入数据"，选择导入方式
3. **清除数据**：进入"设置"页面，点击"清除数据"（谨慎操作）

### 切换UI风格
1. **进入设置页**：点击底部导航栏的"设置"按钮
2. **选择风格**：在"UI风格"中选择喜欢的风格
3. **应用风格**：系统自动应用并保存设置
